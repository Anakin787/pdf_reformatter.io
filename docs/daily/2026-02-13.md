## 상세 작업 내역

### 1. 차트 "+N% 초과" 칩 동적 계산

- **이전:** `+30% 초과` 하드코딩
- **이후:** `numericRate - 15` (현재 문서 표절률 − 제출자 평균 15%) 동적 계산
- 칩 표시 조건도 `value > 30` → `value > avgRate(15)`로 변경
- `measureText()`로 텍스트 너비 측정 → 칩 가로폭 자동 조절

### 2. 결제 링크 클릭 가능 처리 (PAGE 2)

- `{{ paymentLink }}` 텍스트만 표시 → `<a>` 태그로 감싸 클릭 시 새 탭 이동
- 스타일: 파란색(`#60a5fa`) + 밑줄

### 3. PAGE 3 헤더 여백 확대

- "카피클린 디지털 표절 분석 서비스" ↔ "심층 분석 결과지" 사이: `mb-1` → `mb-3`
- "심층 분석 결과지" ↔ 아래 테이블 사이: `mb-2` → `mb-6`

### 4. PAGE 3 Report ID → "자필로 기재하세요"

- Report ID 값을 `{{ reportId }}` 동적 바인딩 → `자필로 기재하세요` 하드코딩
- 스타일: 연한 회색 이탤릭 (성명 필드와 동일)

### 5. 표절률 박스 내부 여백 조정 (PAGE 3)

- "표절률" ↔ 숫자: `mb-1` → `mb-2`
- 숫자 ↔ "판정결과": `mt-2` → `mt-3`
- "판정결과" ↔ 판정 텍스트: `mt-0` → `mt-1`

### 6. 테이블 ↔ 대시보드 여백 확대 (PAGE 3)

- 정보 테이블과 "상세 분석 대시보드" 사이: `mb-4` → `mb-8`

### 7. 카테고리 비율 그래프 색상 통일 (PAGE 3)

- 기존 파랑/초록/주황 개별 색상 → 전부 검은색(`bg-slate-800`)으로 통일

### 8. 대시보드 숫자 색상 변경 (PAGE 3)

- **AI 유사 문장:** 파란색 → 주황색 `#ea580c` (card-ai와 동일)
- **동일한 문장:** 검정 → 빨간색 `text-red-600`
- **표절 의심 문장:** 주황색 → teal `#0d9488` (card-plag와 동일)
- **인용/출처:** 검정 → 회색 `text-slate-400`

### 9. 유사성 분석 분포 바/범례 색상 통일 (PAGE 3)

- 대시보드와 동일 색상 체계 적용
  - 동일 문장: `#22c55e` → `#dc2626` (빨간색)
  - 의심 문장: `#f97316` → `#0d9488` (teal)
  - 인용/출처: `#94a3b8` (회색, 유지)
  - AI 유사: `#3b82f6` → `#ea580c` (주황색)

### 10. 밑줄(`.hl`) 강조 텍스트 볼드 제거 (PAGE 4+)

- `.card-plag .hl`, `.card-ai .hl`: `font-weight: 700` → `font-weight: normal`
