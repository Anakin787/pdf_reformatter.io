## 상세 작업 내역

### 1. GitHub Pages 루트 URL 404 수정

- **증상:** `https://username.github.io/pdf_reformatter/` 접속 시 "File not found" 발생
- **원인:** GitHub Pages는 루트 URL에 `index.html`을 요구하나, 프로젝트에는 `PDF_Reformatter.html`만 존재
- **조치:** 배포 시 `PDF_Reformatter.html`을 `_site/index.html`로 추가 복사
- 결과: `/` 및 `/PDF_Reformatter.html` 모두 정상 접근 가능

### 2. 로고 미표시 수정 (GitHub Pages)

- **증상:** 우측 상단 로고가 완전히 보이지 않음
- **원인:** base64 `originLogoDataUrl` 사용 시 이미지가 정상 렌더링되지 않음
- **조치:**
  - 배포 워크플로에 `origin.png`를 `_site/`로 복사하는 단계 추가
  - 로고 `img` src를 `origin.png`로 변경 (실제 파일 사용)
- 결과: 배포 후 검은 배경 CopyClean 로고 정상 표시

### 3. 2페이지 서비스 상세 견적 프로세스 동적화

- **이전:** AI 유사도 개선 ₩250,000 / 문헌 기반 ₩50,000 / 프로모션 -₩5,000 고정
- **이후:** 사용자가 입력한 `servicePrice`에 맞춰 AI 유사도 개선 항목 동적 계산
  - `pricingBase = servicePrice - 45,000` (문헌 50,000 - 프로모션 5,000)
  - 기본 295,000원 입력 시: 250,000 + 50,000 - 5,000 = 295,000
  - 300,000원 입력 시: 255,000 + 50,000 - 5,000 = 300,000
- `formattedPricingBase` computed 추가, 템플릿에서 동적 바인딩
